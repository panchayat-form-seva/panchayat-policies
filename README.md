## 📱 About Panchayat Form Seva

Panchayat Form Seva is an Android application that helps citizens prepare Panchayat-related application forms quickly and conveniently.

### Features

* User account registration and login
* Fill Panchayat-related forms
* Generate PDF documents
* Save generated PDFs to the user's selected device location
* Simple and user-friendly interface

## 🔒 Privacy

Only the following account information is stored on our servers:

* Username
* Mobile Number
* Email Address
* Address
* Village
* Taluka
* District

Information entered into Panchayat forms is processed only for PDF generation and is **not permanently stored** on our servers.

Generated PDF files are saved only to the location selected by the user on their own device.
